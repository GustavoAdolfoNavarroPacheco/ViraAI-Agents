# Archive

Material antiguo, versiones previas e intermedios pesados que no deben quedar en el flujo principal del proyecto.

## Que Va Aqui

- Versiones antiguas de videos finales.
- Renders intermedios de edicion.
- Sesiones previas de video.
- Material que se conserva por trazabilidad, pero no se usa como fuente activa.

## Que No Va Aqui

- Fuente de verdad institucional (`context/`).
- Skills activas (`skills/`).
- Agentes activos (`.codex/`, `.claude/`).
- Entregables finales vigentes.
